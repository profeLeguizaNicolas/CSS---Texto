CSS - Texto.

A partir de la imagen que se te proporciona, debes desarrollar el documento HTML con sus estilos con CSS:

Tienes que tener en cuenta los siguientes requisitos:

El texto principal:

  1. El color del texto es gris claro: #ddd.
  2. El color de fondo: #111.
  3. El tipo de letra es: cambria.
  4. El tamaño del texto: 16px.
  5. Espacio entre lineas: 1.6

El encabezado de nivel 1 h1:

  1. El color del texto es amarillo: #ffd700.
  2. El tipo de letras es: Calibri.
  3. El tamaño del texto: 30px.
  4. Texto centrado.

El cencabezado de nivel 2, 3 y 4 (h2 h3 h4):

  1. El color del texto es rojo oscuro: #a00.
  2. El tipo de letra es: Calibri.
  3. El tamaño del texto: 24px.
  4. Relleno inferior: 3px

El color de cada item de la sección Datos del libro es amarillo #060 y es font-weight: bolder.

El texto Gotham City, Batimóvil, Robin, Batgirl, Alfred, El Joker, Dos Caras, Acertijo, Murciélago, Caballero Oscuro y Orignes:

  1. El tamaño del texto: 18px.
  2. El color del texto es: #ffd700
  3. Texto en negrita font-weight.

El texto: inteligencia, entrenamiento físico, gadgets, Bruce Wayne, aliados, enemigos y recursos:

  1. El color del texto es: #000.
  2. El color del fondo del texto es: #ffd700.
  3. Texto en italic font-family.
  4. Relleno de izquierda y derecha: 6px.
  5. Relleno de superior y inferior: 2px.
  6. Bordes redondeados de: 4px.
Ayuda:
  • Propiedades que necesitas utilizar.
color - Color de fuente. background-color - Color de fondo. font-family - Estilo de fuente. line-height - Espaciado entre lineas. text-align - Alineación del texto. border-radius - -Redondea los bordes.
